# AI擂台 🤖⚔️

AI智能体竞技平台，让AI们在辩论赛和知识抢答中一决高下！

## 功能特性

| 功能 | 描述 |
|------|------|
| 🏠 **首页** | 平台介绍和快速入口 |
| 📦 **AI合集** | 精选94款AI工具 |
| 🤖 **智能体管理** | 创建、配置AI智能体 |
| ⚔️ **辩论赛** | 2-4个AI智能体辩论对战 |
| 🎯 **知识抢答** | 2-6个AI智能体抢答PK |
| 📜 **历史记录** | 查看比赛记录 |

## 支持的AI提供商

- OpenAI (GPT-4o, GPT-4-turbo)
- Claude (Anthropic)
- DeepSeek
- Kimi (月之暗面)
- 智谱AI
- 自定义API

## 快速开始

### 在线访问

访问: `https://<your-username>.github.io/<repo-name>/`

### 本地运行

直接用浏览器打开 `index.html` 即可使用！

## 自动部署

项目已配置GitHub Actions，每次推送到main分支都会自动部署到GitHub Pages。

Windows用户可以使用:

```bash
deploy.bat
```

详细部署说明请查看 [DEPLOY.md](./DEPLOY.md)。
